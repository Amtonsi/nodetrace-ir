NODETRACE IR — LEGACY-НОСИТЕЛЬ ДЛЯ АНАЛИЗА ЖИВОЙ WINDOWS
========================================================

ВАЖНО: это вспомогательный compatibility-вариант из каталога iso/, а не
основной релиз 0.3.0. Этот конкретный образ НЕ загрузочный и предназначен
только для монтирования в уже запущенной Windows 10/11 или Windows Server.
Настоящий загрузочный x86 WinPE ISO собирается scripts\build_winpe_iso.ps1;
его инструкция находится в winpe\README_RU.txt и корневом README.md.

Запуск
------
1. Проверьте SHA-256 ISO по файлу с расширением .iso.sha256.
2. Смонтируйте ISO штатными средствами Windows.
3. При необходимости откройте проводник/командную строку с правами
   администратора.
4. Запустите START_NODETRACE_IR.cmd вручную.

На носителе НЕТ autorun.inf. Ничего не стартует автоматически при монтировании.
Командный файл проверяет закреплённые размеры и SHA-256 двух неизменённых
официальных ZIP-архивов AVZ, распаковывает их в версионированный доступный для
записи каталог %LOCALAPPDATA%\NodeTraceIR\tool-cache и передаёт приложению:

  NODETRACE_AVZ_ARCHIVE
  NODETRACE_AVZ_BASE_ARCHIVE
  NODETRACE_AVZ_EXE
  NODETRACE_AVZ_CACHE
  NODETRACE_AVZ_DISTRIBUTION=noncommercial-consent

Кейсы сохраняются в %LOCALAPPDATA%\NodeTraceIR\cases. Если база ещё пуста,
приложение автоматически создаёт кейс текущего узла и начинает диагностический
pipeline; отдельной кнопки запуска для этого не требуется.

AVZ не запускается самим командным файлом. Запуск AVZ допускается только из
управляемого диагностического сценария NodeTrace IR, без лечения, удаления,
карантина и иных исправлений системы. Ручной запуск START_NODETRACE_IR.cmd
является явным началом автоматического диагностического pipeline; отдельной
кнопки запуска AVZ в интерфейсе нет.

Состав
------
NodeTraceIR.exe                 приложение расследования
START_NODETRACE_IR.cmd          ручной запуск
AVZ\avz4.zip                    неизменённый архив AVZ
AVZ\avzbase.zip                 неизменённый архив баз AVZ
AVZ_MANIFEST.json               зафиксированные размеры и хэши
MEDIA_SHA256SUMS.txt            SHA-256 исполняемого файла и AVZ-архивов
THIRD_PARTY_AVZ_NOTICE.txt      сведения о стороннем ПО и условиях применения

Ограничения достоверности
-------------------------
Анализ работающей заражённой системы меняет её состояние, а вредоносное ПО
может скрывать или подменять показания. Полную последовательность прошлых
действий нельзя восстановить, если заранее не сохранялись необходимые журналы,
Sysmon/EDR и другие источники телеметрии. Для критичного расследования создайте
образы дисков и памяти и соблюдайте принятую процедуру chain of custody.

ISO является транспортным read-only носителем, но база кейса и версионированный
кэш инструментов создаются в профиле пользователя на доступном для записи диске.
